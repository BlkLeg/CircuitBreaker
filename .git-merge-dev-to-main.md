# Merge dev into main (PR ready)

Your PR is **dev → main**. Main has a few commits dev doesn’t. To make the PR mergeable:

## 1. Fetch and update dev with main

```bash
git fetch origin
git checkout dev
git merge origin/main
```

- If Git says "Already up to date", skip to step 3.
- If there are **conflicts**, fix them in the reported files, then:
  ```bash
  git add .
  git commit -m "Merge main into dev"
  ```

## 2. Push dev

```bash
git push origin dev
```

## 3. Merge the PR on GitHub

Go to the PR and use **Merge pull request** (or merge locally and push main):

```bash
git checkout main
git pull origin main
git merge dev
git push origin main
```

---

**Note:** Your local `main` has 2 commits that `origin/main` doesn’t. If you want local main to match the remote after the PR is merged, run:

```bash
git checkout main
git fetch origin
git reset --hard origin/main
```

(Only do this when you’re sure you don’t need those 2 local commits.)
