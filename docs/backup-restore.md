# Backup & Restore

Use backup and restore to protect your inventory data and recover quickly when needed.

---

## What Backup Includes

A backup captures your inventory snapshot, including your documented infrastructure data and relationships.

---

## Export a Backup

1. Open **Settings**.
2. Go to the system/admin actions area.
3. Choose **Export Backup**.
4. Save the downloaded backup file in a safe location.

Tip: keep date-stamped backups so you can roll back to a known point in time.

---

## Import a Backup

1. Open **Settings**.
2. Choose **Import Backup**.
3. Select your backup file.
4. Confirm restore options.

Use wipe/replace options only when you intentionally want to remove current data before restore.

---

## Clear Lab Data

Clear Lab is a destructive action used to remove current inventory records.

Use this only when you are intentionally resetting your environment.

Best practice:

- Export a fresh backup first.
- Confirm the backup file opens and is valid.

---

## Recovery Checklist

- Keep multiple backups (recent + weekly baseline).
- Store backups off-host when possible.
- Test restore occasionally in a safe environment.
