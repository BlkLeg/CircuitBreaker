# Circuit Breaker v0.1.4 — Release Notes

**Released:** March 4, 2026  
**Previous documented release notes:** v0.1.2

---

## Highlights

v0.1.4 focuses on a stronger daily experience: clearer system controls, broader feature coverage in docs, and improved visibility into how your environment is documented and managed.

---

## Feature Areas in This Release

### Auto-Discovery (Beta)

Discovery is available directly in the app with a full review workflow:

- Scan Profiles for repeatable scans
- Ad-hoc Scan for one-time checks
- Review Queue for controlled approvals
- Scan History for job tracking

### Settings and System Controls

Settings coverage now includes core environment controls such as:

- Language and timezone
- Appearance and branding preferences
- Session and access behavior
- Discovery-related options
- Backup, restore, and clear-lab actions

### Documentation and Guide Coverage

This release includes major documentation refresh work so guidance matches current app behavior and is easier to use day-to-day.

---

## Upgrade Notes

No special migration steps are required for standard upgrades.

If you use protected or sensitive operations, confirm your security-related environment values are set and persisted after upgrade.

---

## Recommended Next Steps After Upgrade

- Verify settings and timezone are correct.
- Run a backup export.
- Review audit history visibility.
- If using discovery, run a small-scope scan first.
